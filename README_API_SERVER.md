# FastAPI 服务器启动说明

## 启动方式

### 方式一：使用启动脚本（推荐）

```bash
python start_server.py
```

### 方式二：使用 uvicorn 命令

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --limit-max-requests 10000 --limit-concurrency 1000 --timeout-keep-alive 300
```

## 重要配置说明

### 大文件上传支持

对于批量上传大量图片（如 10000 张），需要注意：

1. **系统内存**：确保服务器有足够的内存来处理大量文件
2. **网络超时**：前端已配置动态超时时间（最多 2 小时）
3. **批次处理**：后端会自动分批处理，每批 50-100 张图片

### 常见问题

#### 400 Bad Request 错误

如果遇到 400 错误，可能的原因：

1. **文件数量超限**：单次最多支持 10000 张图片
2. **请求格式错误**：检查前端是否正确发送 multipart/form-data
3. **服务器日志**：查看服务器控制台输出的详细错误信息

#### 网络连接重置 (ERR_CONNECTION_RESET)

如果遇到连接重置错误：

1. **请求体太大**：减少单次上传的文件数量
2. **网络不稳定**：检查网络连接
3. **服务器资源不足**：检查服务器内存和 CPU 使用情况

### 调试建议

1. 查看服务器控制台日志，会输出详细的错误信息
2. 查看浏览器控制台，查看具体的错误代码和消息
3. 如果文件数量很多，建议分批上传（每次 1000-2000 张）
